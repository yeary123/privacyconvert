# PWA Icons

Place these files here for "Install app" / Add to desktop to show correct icons:

- **icon-192.png** — 192×192 px (used in taskbar/home screen)
- **icon-512.png** — 512×512 px (used in splash and install dialog)

Use your app logo; keep simple for small sizes. If these files are missing, the site can still be installed but may use a generic icon.
